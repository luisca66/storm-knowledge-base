# Lo que Luis debe aprender

Esta carpeta concentra temas que Luis debe fortalecer para enseñar con mas criterio y evitar promesas exageradas: automatizacion, CRM, procesos empresariales, bases de datos, seguridad, arquitectura de agentes y GitHub profesional.

El objetivo no es convertir cada clase en consultoria tecnica avanzada, sino mejorar el juicio para distinguir entre aprendizaje, prototipo, automatizacion simple y sistema empresarial complejo.
