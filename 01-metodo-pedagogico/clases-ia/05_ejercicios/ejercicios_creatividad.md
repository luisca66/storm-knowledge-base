# Ejercicios: creatividad

## Objetivo

Practicar creatividad con tareas reales y reflexion critica.

## Nivel recomendado

Principiante a intermedio, ajustando complejidad segun alumno.

## Ejercicios para principiante

- Probar una instruccion simple y observar la respuesta.
- Pedir una explicacion en lenguaje sencillo.
- Mejorar el prompt con mas contexto.

## Ejercicios intermedios

- Comparar dos respuestas y elegir la mas util.
- Pedir una version mas clara, breve o profunda.
- Convertir el resultado en una tarea concreta.

## Ejercicio aplicado a caso real

Generar ideas; combinar referencias; prototipar; revisar originalidad y criterio.

## Preguntas para reflexion

- Que parte si sirve?
- Que parte hay que verificar?
- Que dato falto dar?
- Que haria distinto la proxima vez?

## Errores comunes

Pedir algo demasiado amplio, aceptar sin revisar, no guardar aprendizajes o compartir datos sensibles.

## Variantes

Adaptar por edad, proyecto, herramienta disponible y nivel de confianza tecnologica.
