# Leccion 03 - Chat vs agente

## Objetivo

Que el alumno distinga cuando conviene conversar con un chat y cuando conviene usar un agente para ejecutar tareas.

## Para quien sirve

Especialmente para adultos profesionales, creativos, alumnos exploradores y cualquier persona que se confunde entre ChatGPT, Claude, Cowork, Codex, Deep Research, NotebookLM o Antigravity.

## Duracion sugerida

60 a 120 minutos.

## Herramientas

- ChatGPT o Claude.
- Cowork.
- Codex o Claude Code.
- Deep Research.
- NotebookLM.
- Antigravity/Gemini si aplica.

## Conceptos clave

- Chat.
- Agente.
- Investigacion profunda.
- Fuentes.
- Ejecucion.
- Permisos.
- Entregable.

## Filosofia silenciosa

Division de responsabilidad: que delegar y que conservar como juicio humano.

## Preparacion de Luis

Tener una tabla simple de herramientas:

| Necesidad | Mejor punto de partida |
|---|---|
| Pensar o redactar | Chat |
| Investigar panorama | Deep Research |
| Estudiar fuentes | NotebookLM |
| Ejecutar tareas | Agente |
| Crear archivos/proyecto | Codex o Claude Code |
| Explorar prototipo | Antigravity |

## Apertura

Preguntar:

- Quieres pensar, investigar, estudiar, ejecutar o construir?

Esa pregunta decide la herramienta.

## Actividad guiada

Tomar una misma necesidad y pasarla por tres opciones:

1. Chat: aclarar la idea.
2. Deep Research o NotebookLM: buscar o estudiar informacion.
3. Agente: ejecutar entregables.

## Ejercicio practico

El alumno trae una tarea. Luis le pide clasificarla:

- Es conversacion?
- Es investigacion?
- Es estudio de fuentes?
- Es ejecucion?
- Es construccion?

Luego eligen herramienta.

## Preguntas de criterio

- Que datos necesita la herramienta?
- Hay privacidad involucrada?
- Que permisos pide?
- Que salida esperas?
- Como sabras si funciono?

## Variantes por perfil

| Perfil | Variante |
|---|---|
| Carmen | Cotizaciones, foto fija con IA, privacidad |
| Mario | Mapa de herramientas y HTMLs de investigacion |
| Julio | Respaldo cuando se acaba credito |
| Jonas | Que debe hacer el agente vs que debe juzgar el humano |
| Bruno | Una app personal con una sola tarea |

## Errores comunes

- Usar agente para algo que era solo pregunta.
- Usar chat para algo que necesita ejecucion.
- Dar permisos sin revisar privacidad.
- Abrir cinco herramientas sin objetivo.

## Señal de avance

El alumno puede decir: "Para esto uso chat; para esto necesito agente; para esto necesito fuentes."

## Cierre

Crear mapa personal de herramientas del alumno.

## Tarea opcional

Durante la semana, anotar tres tareas y clasificarlas antes de usar IA.
