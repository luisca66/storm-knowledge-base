# Leccion 05 - Aprender y documentar con IA

## Objetivo

Que el alumno aprenda a convertir informacion dispersa en una guia consultable, usando IA para estudiar y documentar sin perder criterio.

## Para quien sirve

Alumnos que necesitan estudiar, investigar, aprender una herramienta, documentar un proceso o construir memoria de proyecto.

## Duracion sugerida

90 a 120 minutos.

## Herramientas

- NotebookLM
- Claude o ChatGPT
- Deep Research
- Codex si se creara estructura de archivos

## Conceptos clave

- Fuente.
- Resumen.
- Guia.
- Pregunta.
- Verificacion.
- Memoria.
- Reutilizacion.

## Filosofia silenciosa

Conocimiento no es acumular informacion. Conocer implica organizar, justificar y poder usar.

## Preparacion de Luis

Tener una fuente simple o pedir al alumno una fuente propia.

## Apertura

Preguntar:

- Que necesitas aprender?
- Que fuentes tienes?
- Que quieres poder hacer despues?

## Actividad guiada

1. Elegir tema.
2. Reunir fuentes.
3. Subir o organizar en NotebookLM.
4. Generar resumen.
5. Generar preguntas.
6. Crear guia.
7. Revisar limites y huecos.

## Ejercicio practico

Crear una guia de aprendizaje de una pagina:

- tema
- para que sirve
- conceptos clave
- pasos
- errores comunes
- preguntas de repaso
- siguiente accion

## Preguntas de criterio

- De donde salio esta informacion?
- Que fuente falta?
- Que parte no entiendes todavia?
- Que pregunta haria alguien que si sabe?
- Que puedes aplicar hoy?

## Variantes por perfil

| Perfil | Variante |
|---|---|
| Estudiante | guia para examen o tema escolar |
| Profesional | guia de proceso de trabajo |
| Creativo | guia de referencias o tecnica |
| Adolescente | guia corta con retos |
| Adulto mayor | guia paso a paso con lenguaje claro |

## Errores comunes

- Confundir resumen con aprendizaje.
- Subir fuentes malas.
- No revisar.
- Crear documentos demasiado largos.
- No definir para que se estudiara.

## Señal de avance

El alumno puede explicar la guia en sus palabras y usarla para hacer algo.

## Cierre

Guardar la guia y definir como se usara antes de la siguiente clase.

## Tarea opcional

Traer una nueva fuente o mejorar la guia con una pregunta propia.

## Siguiente leccion sugerida

Pendiente definir segun perfil: proyecto personal, agentes, privacidad o creatividad.
