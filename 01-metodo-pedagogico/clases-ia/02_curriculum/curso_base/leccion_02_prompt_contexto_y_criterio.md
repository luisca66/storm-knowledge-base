# Leccion 02 - Prompt, contexto y criterio

## Objetivo

Enseñar que un buen resultado de IA depende de dar contexto, definir objetivo y revisar con criterio.

## Para quien sirve

Para cualquier alumno despues de una primera experiencia con IA.

## Duracion sugerida

60 a 90 minutos.

## Herramientas

- ChatGPT
- Claude
- Gemini

## Conceptos clave

- Prompt.
- Contexto.
- Rol.
- Formato.
- Restricciones.
- Iteracion.
- Criterio humano.

## Filosofia silenciosa

Epistemologia basica: no todo lo que la IA dice es conocimiento. Hay que distinguir respuesta, opinion, suposicion y dato verificable.

## Preparacion de Luis

Tener un prompt flojo y un prompt mejorado sobre el mismo tema.

## Apertura

Mostrar dos prompts:

```text
Explícame IA.
```

```text
Explícame IA generativa a una persona de 50 años que trabaja en producción audiovisual, con ejemplos prácticos, riesgos de privacidad y tres usos inmediatos.
```

Preguntar: cual crees que dara mejor resultado y por que?

## Actividad guiada

Construir un prompt con bloques:

1. Tarea.
2. Contexto.
3. Nivel del alumno.
4. Formato.
5. Limites.
6. Criterio de calidad.

## Ejercicio practico

El alumno toma una necesidad propia y la convierte en dos versiones:

- prompt rapido
- prompt con contexto

Despues compara respuestas.

## Preguntas de criterio

- Que cambio entre una respuesta y otra?
- Que contexto ayudo mas?
- Que invento o asumio la IA?
- Que verificarias?
- Que parte puedes usar hoy?

## Variantes por perfil

| Perfil | Variante |
|---|---|
| Adolescente | prompt para app, estudio o juego |
| Profesional | prompt para tarea real de trabajo |
| Creativo | prompt para idea visual, guion o referencia |
| Impaciente | prompt de 5 lineas maximo |
| Adulto mayor | prompt dictado en lenguaje natural |

## Errores comunes

- Confundir prompt largo con prompt bueno.
- Dar informacion sensible innecesaria.
- No pedir formato.
- No revisar si la respuesta cumple la tarea.

## Señal de avance

El alumno puede explicar por que un prompt mejora al agregar contexto y puede detectar al menos una debilidad en la respuesta.

## Cierre

Crear una plantilla personal de prompt para el alumno.

## Tarea opcional

Traer tres tareas reales y convertirlas en prompts con contexto.
