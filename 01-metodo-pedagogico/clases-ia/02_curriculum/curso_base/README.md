# Curso base de IA de Luis

## Proposito

Este curso base contiene lecciones secuenciales y progresivas para guiar las clases personalizadas de IA de Luis.

No es un curso rigido. Es una guia para que Luis no olvide piezas importantes y pueda adaptar cada clase con sensibilidad, criterio e improvisacion.

## Principio central

Pensar primero, usar IA segundo, evaluar despues.

## Filosofia del curso

La IA es el centro visible. La filosofia es el compañero silencioso: criterio, logica, verificacion, autonomia y responsabilidad.

## Uso antes de clase

Si Luis no conoce al alumno, iniciar con Leccion 1. Si ya conoce al alumno, elegir variante segun perfil.

## Uso despues de clase

Registrar bitacora, actualizar expediente del alumno y anotar si la leccion necesita ajustes.

## Regla de actualizacion

Cada leccion debe evolucionar con evidencia de clases reales. Si una actividad no funciona, no se defiende por orgullo: se ajusta.
