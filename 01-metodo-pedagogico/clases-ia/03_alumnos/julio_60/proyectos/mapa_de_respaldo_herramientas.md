# Mapa de respaldo de herramientas

## Objetivo

Ayudar a Julio a saber que usar cuando se acaba credito de Codex, Cowork u otra herramienta.

## Regla simple

Si una herramienta no esta disponible, no detenerse: cambiar a la herramienta mas cercana segun la tarea.

## Mapa inicial

| Si quiero... | Primera opcion | Respaldo |
|---|---|---|
| Pensar o redactar | Claude | ChatGPT |
| Hacer evaluacion rapida | ChatGPT | Claude |
| Prepararme para entrevista | Cowork | Claude con checklist |
| Crear estructura o documento tecnico | Claude Code | Codex si hay credito |
| Trabajar con archivos/proyecto | Claude Code | Codex |
| Ejecutar flujo con tareas | Cowork | Claude paso a paso |

## Checklist anti-desesperacion

1. Estoy en la cuenta correcta?
2. Estoy usando el correo correcto?
3. Tengo credito o uso disponible?
4. La herramienta pide permisos?
5. La tarea esta demasiado grande?
6. Puedo dividirla en una tarea de 10 minutos?
