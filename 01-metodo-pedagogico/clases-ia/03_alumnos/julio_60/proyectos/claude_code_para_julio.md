# Proyecto: Claude Code para Julio

## Objetivo

Introducir Claude Code como siguiente herramienta practica para Julio, aprovechando que le duplicaron el uso.

## Contexto

Julio usa Claude, Cowork y ChatGPT. Necesita herramientas que le ayuden a trabajar menos, resolver rapido y no atascarse cuando se acaba credito de otras plataformas.

## Primer caso ideal

Debe ser una tarea concreta, corta y con salida visible. Ejemplos:

- Organizar una evaluacion financiera en documento.
- Crear una plantilla repetible.
- Revisar un archivo o estructura simple.
- Generar una mini herramienta local.

## Checklist antes de empezar

1. Confirmar cuenta correcta.
2. Confirmar correo correcto.
3. Confirmar credito/uso disponible.
4. Definir una sola tarea.
5. Escribir que resultado quiere.
6. No abrir tres herramientas al mismo tiempo.

## Regla pedagogica

Si algo falla, primero revisar login, correo, cuenta y permisos. No asumir de inmediato que la herramienta no sirve.

## Proximo paso

Hacer una sesion de 45 a 60 minutos maximo con una tarea concreta y documentar el flujo.
