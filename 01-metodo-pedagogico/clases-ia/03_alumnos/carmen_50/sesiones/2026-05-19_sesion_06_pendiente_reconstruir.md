# Bitacora de sesion

- Alumna: Carmen
- Fecha: 2026-05-19
- Duracion: pendiente
- Modalidad: presencial en oficina
- Herramientas usadas: Claude Code, Git en Mac
- Tema central: introduccion a Claude Code con proyecto organizado en carpetas
- Que hicimos: Carmen aprendio a empezar a usar Claude Code. La Mac pidio actualizar una herramienta relacionada con Git, lo cual activo su duda natural por privacidad y seguridad. Luis acompaño la instalacion/actualizacion y despues la dejo trabajando en el proyecto como ella ya entiende: interactuando con carpetas bien nombradas y organizadas.
- Que funciono: aunque Carmen tiene mucho miedo a privacidad, avanzo. La estructura de carpetas ordenadas le dio un marco controlable para interactuar con Claude Code. Al parecer termino contenta con la experiencia y con la forma de trabajar dentro del proyecto.
- Que costo trabajo: la solicitud de actualizacion de Git en la Mac fue un punto de friccion. Para Carmen, cualquier instalacion o permiso nuevo puede disparar sospecha o resistencia.
- Descubrimientos de la alumna: Claude Code puede trabajar dentro de una estructura local organizada sin que la clase se sienta como programacion tradicional. Las carpetas bien nombradas reducen ansiedad porque hacen visible donde esta el trabajo.
- Logro principal: Carmen tuvo su primera interaccion funcional y aparentemente satisfactoria con Claude Code.
- Tarea: pendiente definir siguiente paso del proyecto en Claude Code.
- Proxima clase sugerida: continuar el mismo proyecto en Claude Code, reforzando privacidad, estructura de carpetas, permisos y diferencia entre datos locales, repo y servicios externos.
- Ideas nuevas: convertir la organizacion de carpetas en ritual de seguridad para Carmen antes de usar agentes: que sepa que se comparte, que se queda local, y donde vive cada archivo.
- Actualizaciones para base de datos: registrar Claude Code y Git como herramientas vistas por Carmen; registrar avance por tolerar instalacion/actualizacion y trabajar con proyecto local.
- Observaciones de Luis: Carmen es muy chistosa y tiene panico de privacidad, pero sigue avanzando. Su confianza se construye con explicaciones claras, limites visibles y estructura. El contexto operativo mencionado por Luis fue que Anthropic habia ampliado el uso de Claude Code, razon por la cual convenia aprovecharlo; verificar fuente antes de convertirlo en afirmacion documentada del curso.

## Contexto previo

Carmen venia trabajando con agentes, Cowork, Claude, Deep Research y Codex, especialmente alrededor de cotizaciones, investigacion profesional y documentacion de procesos creativos. Antes de esta sesion, el pendiente pedagogico era reforzar cuando usar chat, cuando usar agente, como cuidar privacidad y como convertir el aprendizaje de foto fija con IA en un sistema consultable.

## Hallazgo pedagogico

Para Carmen, la entrada a herramientas de codigo no debe presentarse como "programar", sino como trabajar con un proyecto local ordenado. La estructura de carpetas funciona como puente emocional: reduce la incertidumbre, permite explicar privacidad y convierte a Claude Code en una herramienta de organizacion/produccion antes que en una herramienta tecnica.
