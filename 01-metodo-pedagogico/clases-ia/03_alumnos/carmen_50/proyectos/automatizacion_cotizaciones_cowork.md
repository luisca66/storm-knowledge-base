# Proyecto: automatizacion de cotizaciones con Cowork

## Alumna

Carmen, junto con Mario, Montse y Karla.

## Objetivo

Usar Cowork para apoyar la preparacion de cotizaciones y entregables relacionados con produccion creativa.

## Herramientas

- Cowork
- Claude

## Flujo descubierto

1. Claude actua como entrevistador.
2. Claude genera un cuestionario para obtener la informacion necesaria.
3. Carmen, Mario o el equipo responden.
4. Con esas respuestas se arma un prompt mas claro.
5. Cowork recibe el prompt y genera entregables o estructura de trabajo.

## Lo que funciono

El cuestionario previo evita prompts vagos. El equipo no tiene que saber escribir el prompt perfecto desde cero; puede responder preguntas guiadas y convertir experiencia profesional en instrucciones para el agente.

## Lo que falta

- Definir plantillas de cotizacion.
- Identificar datos sensibles que no deben compartirse.
- Crear checklist de permisos.
- Medir si Cowork ahorra tiempo real frente al proceso manual de Carmen.
- Separar tareas donde IA ayuda de tareas donde la experiencia humana sigue siendo mas rapida.

## Riesgos

- Privacidad.
- Permisos excesivos.
- Automatizar mal un proceso que requiere criterio fino.
- Tardar mas en configurar que en cotizar manualmente.

## Proximo paso

Crear una prueba con un caso ficticio o anonimizado para medir:

- tiempo manual estimado
- tiempo con Claude + Cowork
- calidad del entregable
- datos sensibles involucrados
- decision final: sirve, no sirve o sirve solo para ciertos casos
