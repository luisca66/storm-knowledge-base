# Reporte inicial para Luis - Carmen

## Estado de Carmen

Carmen es una alumna profesional creativa con aproximadamente cinco clases. Tiene nivel tecnologico medio y ya esta usando Cowork y Claude. Su necesidad no es aprender IA desde cero, sino entender como usarla profesionalmente sin perder control, privacidad ni criterio.

## Que esta funcionando

- Casos reales de negocio.
- Ver que IA puede llegar a conclusiones utiles.
- Usar Claude para estructurar preguntas antes de pedir entregables.
- Comparar herramientas como Deep Research y Codex.

## Que se debe reforzar

- Diferencia entre chat y agente.
- Privacidad antes de permisos.
- Medicion de ahorro real de tiempo.
- Uso de IA como apoyo al criterio experto, no como sustituto.

## Que evitar

- Codigo por ahora.
- Flujos con permisos excesivos.
- Automatizacion demasiado ambiciosa.
- Prometer que la IA hara mejor todo lo que Carmen ya hace rapido por experiencia.

## Proxima clase recomendada

Proyecto "foto fija con IA":

1. Claude entrevista a Carmen.
2. Se define que necesita aprender.
3. Se decide herramienta: NotebookLM, Cowork, Codex o Antigravity.
4. Se crea una primera guia consultable.

## Hallazgo para curso base

Para adultos profesionales, una leccion temprana debe ser: "chat vs agente". Si no entienden esa diferencia, pueden usar herramientas demasiado pesadas para tareas simples o chats simples para tareas que requieren ejecucion organizada.
