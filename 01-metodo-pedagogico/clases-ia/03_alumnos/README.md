# Sistema de alumnos

Cada alumno tiene una carpeta propia con perfil, diagnostico, sesiones, tareas, proyectos, reportes internos y notas privadas.

No se deben inventar datos. Si algo falta, se escribe `pendiente`. El perfil se actualiza cuando haya informacion estable. Las sesiones se registran en archivos fechados y se conectan con los CSVs de `09_base_de_datos`.

Las notas privadas deben usarse con cuidado y solo cuando ayuden a Luis a enseñar mejor.
