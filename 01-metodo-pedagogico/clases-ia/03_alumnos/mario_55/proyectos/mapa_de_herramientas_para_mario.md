# Mapa de herramientas para Mario

## Objetivo

Ayudar a Mario a recordar que herramienta conviene para cada tipo de tarea.

## Regla simple

Primero definir la tarea, despues elegir herramienta. No abrir herramientas por impulso.

## Mapa inicial

| Necesidad | Herramienta sugerida | Por que |
|---|---|---|
| Pensar una idea, aclarar objetivo o preparar instrucciones | Claude o ChatGPT | Buenas para conversar, estructurar y preguntar |
| Generar un cuestionario antes de un proyecto | Claude | Funciona bien como entrevistador |
| Investigar mercado o tendencias con profundidad | Deep Research o Codex | Sirven para comparar informacion y producir reportes |
| Ejecutar un flujo de trabajo con entregables | Cowork | Mejor cuando hay tareas, pasos y archivos |
| Crear documentos, estructura o HTML consultable | Codex | Bueno para generar repos, docs y archivos |
| Diseñar o explorar una interfaz visual | Claude Design | Bueno para visual y web conceptual |
| Estudiar fuentes guardadas | NotebookLM | Bueno para aprender desde fuentes concretas |
| Explorar con Gemini o prototipos asistidos | Antigravity con Gemini | Pendiente probar |
| Imagen y video | pendiente | Preguntar que usa Mario actualmente |

## Preguntas antes de elegir herramienta

1. Quiero pensar, investigar, ejecutar, diseñar o estudiar?
2. Tengo fuentes o necesito buscarlas?
3. Hay datos sensibles?
4. Necesito un entregable o solo orientacion?
5. Lo quiero hacer una vez o repetirlo como proceso?

## Proximo uso en clase

Aplicar este mapa a un caso real de Mario y pedirle que elija herramienta antes de abrirla. Despues comparar si la eleccion funciono.
