# Proyecto: app de entrenamiento para iPhone

## Alumno

Bruno.

## Objetivo

Crear o instalar una app de entrenamiento que Bruno pueda usar en su iPhone para registrar o guiar su trabajo fisico.

## Contexto

Bruno tiene un interes fuerte por el gym y la carga pesada. A los 15 años hizo bench press de 148 kg 1RM. Este interes es una puerta pedagogica para conectar disciplina fisica, atencion y uso practico de IA.

## Herramientas posibles

- Codex
- Claude
- Claude Code
- Gemini
- Web app instalable como PWA
- Herramientas nativas de iPhone si se resuelve el flujo tecnico

## Estado actual

Pendiente instalar o dar de alta la app de entrenamiento como app nativa en iPhone. El bloqueo principal son los candados y restricciones del ecosistema Apple.

## Riesgos

- Consumir demasiada clase resolviendo temas tecnicos de Apple.
- Frustrar a Bruno con pasos administrativos.
- Perder el foco pedagogico de atencion y uso de IA.

## Alternativas

- Hacer una PWA simple que pueda abrir desde Safari.
- Crear una version web responsive.
- Usar un prototipo local en la computadora de Luis y pasar a iPhone despues.
- Crear una app nueva mas simple enfocada en atencion o registro basico.

## Proximo paso

Definir la ruta mas simple:

1. App nativa si el flujo ya esta claro.
2. PWA si se busca rapidez.
3. Prototipo web si el objetivo es aprendizaje y no instalacion.
