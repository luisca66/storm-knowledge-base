# Criterios de calidad de clase

Una clase fue buena si el alumno participo, hubo descubrimiento practico, se resolvio o avanzo un problema real, se registro la sesion, quedo una tarea clara, se detecto el siguiente paso y el alumno gano autonomia.

Tambien cuenta si Luis aprendio algo sobre como explicar mejor, que resistencia atender o que ruta probar la proxima vez.
