# Kit de clase - 2026-05-19 - Leccion 01

## Proposito

Este kit prepara la primera clase con alumnas nuevas cuando Luis no sabe casi nada de ellas.

La meta no es cubrir teoria. La meta es lograr una primera victoria con IA, diagnosticar perfil y cerrar con siguiente paso claro.

## Archivos del kit

- `guion_de_clase.md`: secuencia de 2 horas.
- `preguntas_diagnostico_rapido.md`: preguntas para conocer a la alumna sin volverlo interrogatorio.
- `prompts_para_primera_victoria.md`: prompts listos para usar en vivo.
- `rutas_de_emergencia.md`: que hacer segun el tipo de alumna que aparezca.
- `checklist_de_cierre.md`: cierre obligatorio para no perder memoria.
- `bitacora_lista_para_copiar.md`: formato posterior a la clase.

## Regla de uso

Abrir primero `guion_de_clase.md`.

Durante la clase, elegir una sola ruta. No intentar enseñar todas las herramientas.

Despues de la clase, llenar bitacora y actualizar expediente.
