# Rutas de emergencia

## Si la alumna no sabe que quiere

Usar ruta de organizacion:

1. Hacer lista de 10 pendientes o intereses.
2. Elegir el mas molesto o util.
3. Convertirlo en prompt.
4. Salir con plan de 3 pasos.

## Si tiene miedo o desconfianza

Usar ruta de criterio:

1. Pedir a IA una respuesta simple.
2. Revisarla juntos.
3. Encontrar supuestos o datos no verificados.
4. Mostrar que Luis no pide fe; pide criterio.

## Si quiere algo profesional

Usar ruta de trabajo:

1. Elegir una tarea real.
2. Separar datos sensibles.
3. Pedir ayuda sin compartir informacion delicada.
4. Convertir en plantilla reusable.

## Si quiere algo creativo

Usar ruta de tres opciones:

1. Pedir ruta conservadora, creativa y ambiciosa.
2. Comparar.
3. Elegir una.
4. Pedir primer borrador.

## Si quiere estudiar

Usar ruta guia:

1. Tema.
2. Nivel.
3. Guia breve.
4. Preguntas.
5. Plan de estudio.

## Si se abruma

Reducir:

- una herramienta
- una pregunta
- una salida
- una accion

Frase util:

"No tenemos que entender toda la IA hoy. Solo vamos a lograr una cosa util y revisar que tan confiable fue."

## Si pide agentes demasiado pronto

Explicar:

"Primero usamos chat para pensar. Si la tarea necesita ejecutar varios pasos o producir archivos, entonces pasamos a agente."

## Si pide codigo

Si es principiante:

"Hoy no vamos a aprender codigo. Podemos crear algo con IA, pero el objetivo es entender que pedir y como revisar."

Si tiene perfil tecnico:

"Podemos hacer una mini construccion, pero cerramos con que hizo la IA, que hiciste tu y que falta verificar."

## Si aparece privacidad

Pausar y hacer checklist:

- Hay datos personales?
- Hay clientes?
- Hay contraseñas?
- Hay informacion financiera?
- Podemos anonimizar?
- Podemos usar ejemplo ficticio?
