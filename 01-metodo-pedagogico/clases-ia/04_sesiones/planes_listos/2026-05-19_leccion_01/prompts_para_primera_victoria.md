# Prompts para primera victoria

## Prompt universal

```text
Actua como asistente practico y claro.

Quiero lograr: [objetivo].
Mi contexto: [contexto].
Mi nivel con este tema: [nivel].
Restricciones: [privacidad, tiempo, formato, tono].

Antes de responder, hazme hasta 5 preguntas si falta informacion importante.
Si puedes responder con lo que ya tienes, entrega una respuesta breve, util y accionable.
No inventes datos. Marca como pendiente lo que falte.
```

## Organizar una idea confusa

```text
Tengo esta idea desordenada:
[pegar idea]

Ayudame a convertirla en:
1. objetivo claro
2. pasos concretos
3. dudas pendientes
4. primer siguiente paso de menos de 15 minutos

No agregues cosas que no se desprendan de lo que escribi.
```

## Crear una guia de estudio

```text
Quiero estudiar este tema:
[tema]

Mi nivel actual es:
[nivel]

Crea una guia de estudio breve con:
- explicacion simple
- 5 conceptos clave
- 5 preguntas de repaso
- errores comunes
- plan de estudio de 3 dias

Si falta informacion, preguntame primero.
```

## Mejorar una tarea de trabajo

```text
Tengo esta tarea de trabajo:
[tarea]

Ayudame a mejorarla con IA.
Necesito:
- que partes puede ayudar a hacer la IA
- que partes debo decidir yo
- riesgos de privacidad
- prompt inicial
- resultado esperado
```

## Crear una primera idea visual o creativa

```text
Quiero crear una idea para:
[proyecto creativo]

Dame 3 rutas distintas:
1. una conservadora
2. una creativa
3. una ambiciosa

Para cada ruta incluye:
- descripcion
- herramientas posibles
- riesgos
- primer paso
```

## Revisar respuesta de IA

```text
Revisa esta respuesta de IA:
[pegar respuesta]

Evaluala con estos criterios:
- que si sirve
- que podria estar inventado
- que falta verificar
- que parte esta demasiado general
- como pedirias una mejor version
```

## Convertir salida en accion

```text
Convierte la respuesta anterior en un plan de accion de 3 pasos.

Cada paso debe tener:
- accion concreta
- tiempo estimado
- herramienta sugerida
- resultado esperado

Manténlo simple y realista.
```

## Cierre de clase

```text
Con base en lo que hicimos hoy, resume:
1. que queria lograr la alumna
2. que herramienta usamos
3. que resultado obtuvo
4. que aprendio sobre IA
5. que debe revisar con criterio
6. que tarea pequena queda para la siguiente clase

No inventes datos. Si algo no se dijo, marca pendiente.
```
