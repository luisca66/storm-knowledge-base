# Contexto del proyecto

Luis desarrolla clases personalizadas de inteligencia artificial para alumnos de edades y perfiles distintos. Las clases son presenciales y en linea, normalmente de dos horas, con un precio objetivo de 2000 MXN por clase por persona.

Luis se identifica como vibe coder: no es un programador tradicional, pero usa IA, agentes y herramientas modernas para resolver problemas reales, construir proyectos y aprender haciendo. Esa experiencia es una ventaja pedagogica porque le permite acompanar a personas que tambien empiezan desde niveles bajos o medios.

## Que ensena

Luis ensena uso practico de IA para aprender, crear, organizarse, estudiar, producir ideas, trabajar con agentes y transformar problemas reales en proyectos concretos. Las herramientas contempladas incluyen ChatGPT, Claude, Gemini, NotebookLM, ElevenLabs, Suno, Codex, GitHub, Antigravity, Claude Code, Cowork y automatizacion basica.

## Como ensena

El metodo es adaptativo. La clase parte de intereses, problemas y ritmo del alumno. No busca memorizar funciones, sino desarrollar criterio, autonomia y pensamiento critico. La improvisacion es bienvenida, pero debe registrarse para mejorar el sistema.

## Tipo de alumnos

El sistema contiene expedientes para 11 alumnos: Jonas, Bruno, Esteban, Karla, Montse, Mariana, Dayana, Carmen, Mario, Julio y Luis (Papa). Los perfiles tienen distintos grados de avance; todo dato faltante debe marcarse como `pendiente` y completarse mediante diagnostico o reconstruccion con Luis.

## Limites

Luis puede ofrecer acompanamiento practico, organizacion, introduccion a agentes y productividad con IA. No debe prometer infraestructura empresarial avanzada, seguridad compleja, bases de datos profesionales ni sistemas criticos sin apoyo tecnico especializado.

## Estado actual

Sistema integrado al Knowledge Base de Storm Studios Learning. Base en Markdown y CSV con 11 expedientes de alumnos; perfiles, diagnosticos y sesiones tienen distintos grados de avance.
