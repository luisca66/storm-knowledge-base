# Privacidad y datos sensibles

## Datos que se pueden registrar

Nombre, edad, objetivos de aprendizaje, herramientas vistas, tareas, avances, dificultades pedagogicas, proyectos de clase y observaciones necesarias para mejorar el aprendizaje.

## Datos que conviene evitar

Informacion medica, familiar, financiera, contraseñas, datos bancarios, documentos oficiales, ubicaciones sensibles y detalles personales que no aporten al aprendizaje.

## Anonimizacion

Si una nota pudiera exponer al alumno, usar iniciales, descripciones generales o moverla a `notas_privadas` con lenguaje prudente.

## Notas personales

Registrar solo lo necesario para mejorar la clase. Evitar juicios personales. Escribir observaciones como hipotesis pedagogicas, no como etiquetas fijas.

## Reglas

- No subir datos delicados a repositorios publicos.
- Mantener el repositorio privado si contiene datos reales.
- No compartir reportes internos sin revisar contenido.
- No pegar informacion sensible en herramientas externas innecesariamente.
