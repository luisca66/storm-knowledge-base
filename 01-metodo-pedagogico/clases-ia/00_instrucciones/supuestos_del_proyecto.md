# Supuestos del proyecto

- Solo se conocen nombres y edades de los alumnos actuales.
- Las clases duran normalmente dos horas, pero cada registro puede ajustar la duracion real.
- Los reportes son internos para Luis, no documentos para enviar al alumno.
- El repositorio debe mantenerse privado si contiene datos reales.
- Markdown y CSV son suficientes para iniciar sin complejidad tecnica.
- El modulo empresarial debe ser introductorio hasta que Luis fortalezca CRM, automatizacion, bases de datos y seguridad.
- Las fuentes importadas deben normalizarse a nombres limpios cuando sea razonable; si hacerlo rompe indices internos complejos, crear al menos mapas pedagogicos nuevos con nombres limpios.
