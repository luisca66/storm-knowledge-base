# Antes de cada clase

- Revisar carpeta del alumno.
- Leer ultima bitacora disponible.
- Revisar tareas pendientes.
- Ver herramientas ya vistas.
- Proponer 2 o 3 rutas posibles.
- Preparar un objetivo flexible.
- Tener un plan alternativo.
- Identificar una idea practica para enganchar al alumno.
- Revisar riesgos, bloqueos o resistencias.
- Preparar una pregunta inicial para conectar con el interes real del alumno.
