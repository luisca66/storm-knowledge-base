# Instrucciones generales para agentes

Leer primero `README.md`, `PROJECT_CONTEXT.md`, `AGENTS.md`, `vision_del_proyecto.md`, `ultimo_estado.md` y `pendientes.md`.

Identificar si la tarea pertenece a alumnos, clases, metodo, curriculum, ejercicios, prompts, fuentes, base de datos, reportes o aprendizaje de Luis.

No hacer preguntas salvo que sea indispensable. Si falta un dato, usar `pendiente`, tomar un supuesto razonable y documentarlo en `supuestos_del_proyecto.md`.

Mantener todo en español. Usar archivos sin acentos ni espacios. Registrar cambios relevantes en `ultimo_estado.md` y, si aplica, en `bitacora_global.csv`.

Cuando se creen nuevas sesiones o materiales, actualizar los registros relacionados para que el repositorio siga siendo consultable.

Cuando se trabaje en el curso base, priorizar lecciones secuenciales, progresivas y reutilizables. El agente debe ayudar a detectar tendencias, revisar bitacoras incompletas y sugerir mejoras para las lecciones.
