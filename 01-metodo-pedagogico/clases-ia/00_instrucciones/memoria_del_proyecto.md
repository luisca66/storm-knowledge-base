# Memoria del proyecto

Luis enseña IA de forma adaptativa. El objetivo central es que el alumno gane autonomia para aprender, resolver y crear con IA.

El proyecto ayuda a planear, registrar y mejorar clases personalizadas. Los reportes son de uso interno para Luis. Este sistema esta separado de Storm Studios Learning.

La base inicial usa Markdown para conocimiento cualitativo y CSV para datos ligeros. Se podra agregar JSON cuando convenga, especialmente para automatizaciones o integraciones futuras.

El pensamiento critico es parte central del metodo: la IA no se presenta como magia, sino como herramienta poderosa con limites, sesgos y responsabilidades.
