# Migracion Empresas

## Resumen

Migracion Empresas es un nuevo proyecto de Luis orientado al trabajo con empresas mediante IA. Ya es una linea activa y esta generando ingresos.

Este archivo es la fuente canonica para documentar su identidad, propuesta, modelo comercial, metodo de trabajo y relacion con los proyectos de cada empresa. Los detalles tecnicos de cada implementacion viven en sus repositorios propios.

## Lo confirmado

- Es un proyecto nuevo de Luis.
- Ya produce entradas de dinero.
- Agrupa proyectos empresariales con repositorios independientes.
- Voces Imaginarias vive dentro de Migracion Empresas y tiene su propio proyecto y repo.
- Las clases de IA pueden usar esos proyectos como contexto pedagogico, pero no son su fuente de verdad.
