# Mapa de tendencias IA para clases

## Fuente base

Resumenes diarios de IA guardados por mes en `08_fuentes/noticias_ia_diarias/`.

Meses disponibles:

- `202603`: marzo de 2026, con indice general.
- `202604`: abril de 2026, con indice general.
- `202605`: mayo de 2026, con resumenes del 1 al 4 de mayo.

## Valor para el proyecto

Estas noticias ayudan a Luis a mantener las clases vivas y conectadas con el presente. No deben usarse como aval automatico de verdad; deben tratarse como material de contexto, conversacion, pensamiento critico y deteccion de oportunidades.

## Tendencias detectadas

| Tendencia | Como puede aparecer en clase | Precaucion |
|---|---|---|
| Agentes y flujos agenticos | Mostrar que la IA ya no solo responde; tambien ejecuta tareas | Enseñar supervision humana y revision |
| Codex, Claude Code y herramientas de construccion | Introducir al alumno a proyectos asistidos | No prometer sistemas profesionales sin base tecnica |
| Costos, inferencia y limites fisicos | Explicar que la IA tiene costos reales | Evitar vender la IA como recurso infinito |
| Empresas AI-native vs empresas tradicionales | Comparar cultura de adopcion y bloqueos | No simplificar problemas de gobernanza |
| Privacidad, seguridad y shadow IT | Conversar sobre datos sensibles y herramientas no aprobadas | Cuidar ejemplos reales de alumnos |
| Mercado laboral y habilidades | Motivar autonomia y criterio de aprendizaje | Evitar alarmismo paralizante |
| Memoria institucional y conocimiento persistente | Reforzar la importancia de este repositorio | No depender de memoria temporal del chat |
| Skills, MCP y herramientas para agentes | Preparar rutas futuras para alumnos avanzados | Introducir solo cuando haya base suficiente |

## Ideas para clases

- "La IA como amplificador de criterio": comparar una herramienta general y una especializada para una misma tarea.
- "No todo agente debe correr solo": diseñar una tarea con punto de revision humana.
- "Costo oculto de una mala herramienta": medir tiempo, calidad y utilidad en una tarea semanal.
- "Herramienta por defecto vs herramienta correcta": ayudar al alumno a elegir segun trabajo real.
- "Repositorio como memoria": mostrar por que guardar bitacoras, fuentes y decisiones importa.

## Preguntas de pensamiento critico

- Que evidencia trae esta noticia?
- Que podria ser exageracion o marketing?
- A quien beneficia esta narrativa?
- Que riesgo practico se desprende para alumnos o empresas?
- Que habilidad deberia aprender Luis o el alumno a partir de esto?

## Uso recomendado

Antes de una clase, elegir una noticia o tendencia maxima. Convertirla en una pregunta, ejercicio o ejemplo. Despues de clase, registrar si funciono y que ajustes pedagogicos genero.
